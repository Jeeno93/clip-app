Создай функцию:

function smartTruncate(text: string, limit: number): string {
  if (text.length <= limit) return text
  
  // Берём 70% начала и 30% конца
  // Начало содержит введение и главные тезисы
  // Конец содержит выводы и заключение
  const startPart = Math.floor(limit * 0.7)
  const endPart = Math.floor(limit * 0.3)
  
  const start = text.slice(0, startPart)
  const end = text.slice(-endPart)
  
  // Обрезаем по границам слов
  const startClean = start.slice(0, start.lastIndexOf(' '))
  const endClean = end.slice(end.indexOf(' ') + 1)
  
  return startClean + 
    '\n\n[...фрагмент статьи пропущен...]\n\n' + 
    endClean
}

ШАГ 4 — Улучшить парсинг в fetchLinkPreview.ts
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

В функции extractFullText улучши извлечение контента:

1. Для Хабра — специфичный селектор:
   Хабр хранит контент в div.article-formatted-body
   Ищи паттерн: class="article-formatted-body"
   или class="tm-article-body"

2. Общий порядок поиска контента:
   1) div с классом содержащим "article-body" или 
      "article-content" или "post-body" или "entry-content"
   2) <article> тег
   3) <main> тег  
   4) Весь body

3. Более агрессивная очистка мусора — убирай:
   - Все теги: script, style, nav, header, footer, 
     aside, figure, form, iframe, svg
   - Элементы с классами содержащими: 
     "comment", "sidebar", "widget", "banner", 
     "advertisement", "social", "share", "related"
     (через regex на class и id атрибуты)

4. Декодирование HTML entities более полное:
   &nbsp; → пробел
   &amp; → &
   &lt; → 
   &gt; → >
   &quot; → "
   &#39; → '
   &mdash; → —
   &ndash; → –
   &laquo; → «
   &raquo; → »
   &#[0-9]+; → соответствующий символ через 
     String.fromCharCode()

5. fetchLinkPreview — увеличь таймаут до 10 секунд
   Добавь заголовки для лучшей совместимости:
   headers: {
     'User-Agent': 'Mozilla/5.0 (compatible; ClipBot/1.0)',
     'Accept': 'text/html,application/xhtml+xml',
     'Accept-Language': 'ru,en;q=0.9',
   }
