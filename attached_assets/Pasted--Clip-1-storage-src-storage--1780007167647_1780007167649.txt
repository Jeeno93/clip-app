Улучши онбординг в Clip и добавь демо-карточки.

ШАГ 1 — Демо-карточки в storage

В src/storage/clips.ts добавь функцию:

async function addDemoClips(): Promise<void>

Функция добавляет 5 демо-карточек если их ещё нет
(проверка по тегу #demo):

const existing = await getAllClips()
const hasDemo = existing.some(c => c.tags.includes('demo'))
if (hasDemo) return

for (const demo of demoClips) {
  await saveClip({
    ...demo,
    createdAt: new Date().toISOString(),
  })
}

ШАГ 2 — Автодобавление при первом запуске

В app/onboarding.tsx (или где хранится логика 
первого запуска) — после завершения онбординга 
вызвать addDemoClips().

Если онбординг уже пройден (старые пользователи) —
не добавлять автоматически.

ШАГ 3 — Кнопка в настройках

В app/settings.tsx в раздел "Данные" добавь кнопку:

"✦ Добавить демо-карточки"

Стиль: как кнопка экспорта (bgCard, border, 
borderRadius md).
Подпись: "Посмотри как работает Clip на примерах"
textMuted fontSize xs.

При нажатии:
- Если демо уже есть — Alert 
  "Демо-карточки уже добавлены. 
   Найди их по тегу #demo в архиве."
- Если нет — addDemoClips() + Alert 
  "Добавлено 5 демо-карточек. 
   Найди их по тегу #demo в архиве."
