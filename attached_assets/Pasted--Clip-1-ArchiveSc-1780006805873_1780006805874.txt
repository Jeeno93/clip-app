ШАГ 1 — Режим выбора в ArchiveScreen

В app/(tabs)/archive.tsx добавь:

Стейт:
const [selectionMode, setSelectionMode] = useState(false)
const [selectedIds, setSelectedIds] = useState<Set<string>>(new Set())

Вход в режим выбора:
- Long press на любую карточку → selectionMode = true, 
  карточка добавляется в selectedIds

Выход из режима выбора:
- Кнопка "Отмена" в шапке → selectionMode = false, 
  selectedIds очищается

for (const id of selectedIds) {
  const clip = clips.find(c => c.id === id)
  if (clip) {
