ШАГ 2 — Поле contentTypeId в Clip

В структуру Clip добавь:
  contentTypeId?: string  // null = не выбран

ШАГ 3 — Выбор типа в AddClipScreen

В app/add.tsx добавь секцию "ТИП КОНТЕНТА" 
между полем текста и тегами:

Горизонтальный скролл чипов (как было у источника):
[🔬 Исследование] [📖 Туториал] [📰 Новость] 
[📚 Книга] [💬 Пост] [📄 Другое] [+ Свой тип]

Активный чип: accentSubtle фон, accent граница
Неактивный: bgTag фон, border граница

При сохранении — записывать contentTypeId в карточку.

Кнопка "+ Свой тип" открывает модал создания 
нового типа (см. ШАГ 5).

ШАГ 4 — Адаптация промпта в summarize.ts

В src/utils/summarize.ts добавь параметр:
contentTypeHint?: string

В системный промпт добавить если передан:
`Тип материала: ${contentTypeHint}`

В app/clip/[id].tsx перед вызовом summarizeContent:
const contentType = allContentTypes.find(
  t => t.id === clip.contentTypeId
)
const contentTypeHint = contentType?.promptHint

Передавать contentTypeHint в summarizeContent.

ШАГ 5 — Модал создания своего типа
(src/components/CreateContentTypeModal.tsx)

Modal снизу.
- Выбор иконки (те же эмодзи что у доменов)
- TextInput названия типа (maxLength: 30)
- Кнопка "Создать" → saveCustomContentType()
- Кнопка "Отмена"

ШАГ 6 — Экран справочника типов контента
(app/content-types.tsx)

Список всех типов: встроенные + пользовательские.

Встроенные:
- Иконка + название
- Серая пометка "встроенный"
- Нельзя удалить

Пользовательские:
- Иконка + название
- Кнопка удаления (корзина) справа
- Свайп влево для удаления

Кнопка "+ Добавить свой тип" внизу.

Вход в справочник: в app/settings.tsx 
добавь кнопку "Типы контента →" 
в раздел рядом со справочником тегов.
