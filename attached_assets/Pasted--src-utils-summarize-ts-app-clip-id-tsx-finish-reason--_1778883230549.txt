ШАГ 1 — Возвращать finish_reason из summarize.ts

Измени возвращаемый тип функции summarizeContent:

БЫЛО:
async function summarizeContent(...): Promise<string | null>

СТАЛО:
interface SummarizeResult {
  text: string
  truncated: boolean  // true если finish_reason === "length"
}

async function summarizeContent(...): 
  Promise<SummarizeResult | null>

Для каждого провайдера извлекай finish_reason:

Gemini:
const finishReason = data.candidates?.[0]?.finishReason
const truncated = finishReason === "MAX_TOKENS"

Claude:
const finishReason = data.stop_reason
const truncated = finishReason === "max_tokens"

OpenAI и DeepSeek:
const finishReason = data.choices?.[0]?.finish_reason
const truncated = finishReason === "length"

YandexGPT:
const finishReason = 
  data.result?.alternatives?.[0]?.status
const truncated = 
  finishReason === "ALTERNATIVE_STATUS_TRUNCATED_FINAL"

Возвращать: { text: extractedText, truncated }
При ошибке — возвращать null как раньше.

ШАГ 2 — Добавить overrideMaxTokens параметр

Добавь опциональный параметр в summarizeContent:
overrideMaxTokens?: number

Если передан — использовать его вместо 
getMaxTokens(depth, modules).

ШАГ 3 — Обработка в ClipDetailScreen

В app/clip/[id].tsx обнови handleAnalyze:

const result = await summarizeContent(...)

if (!result) {
  Alert.alert('Ошибка AI', error?.message || 
    'Не удалось получить ответ')
  return
}

// Сохраняем результат
await updateClip(id, { 
  summary: result.text,
  summaryTruncated: result.truncated  // новое поле
})
setClip(prev => ({ 
  ...prev, 
  summary: result.text,
  summaryTruncated: result.truncated
}))

// Если обрезан — показываем предложение
if (result.truncated) {
  const currentMax = getMaxTokens(
    settings.aiDepth, 
    localModules ?? settings.aiModules
  )
  const extendedMax = Math.min(currentMax * 2, 16000)
  
  Alert.alert(
    'Анализ был обрезан',
    `Материал длиннее стандартного лимита. Повторить анализ с увеличенным лимитом? Это потребует примерно вдвое больше токенов.`,
    [
      { 
        text: 'Повторить', 
        onPress: () => handleAnalyze(extendedMax)
      },
      { 
        text: 'Оставить как есть', 
        style: 'cancel' 
      }
    ]
  )
}
