Политика конфиденциальности Clip
Последнее обновление: май 2026

СБОР ДАННЫХ
Clip не собирает, не хранит и не передаёт персональные данные 
пользователей. Приложение не требует регистрации или входа 
в аккаунт.

ХРАНЕНИЕ ДАННЫХ
Все данные (идеи, теги, домены, настройки) хранятся исключительно 
на устройстве пользователя. Данные не синхронизируются 
с серверами разработчика.

API КЛЮЧИ
Если пользователь вводит API ключ стороннего сервиса — ключ 
хранится локально и используется только для прямых запросов 
к соответствующему сервису. Разработчик Clip не имеет доступа 
к API ключам пользователей.

СТОРОННИЕ СЕРВИСЫ
При использовании AI-анализа приложение отправляет текст статьи 
напрямую в выбранный AI-сервис. На эти запросы распространяется 
политика конфиденциальности соответствующего сервиса.
При загрузке превью ссылок приложение делает запрос к указанному 
URL для получения метаданных страницы.

РАЗРЕШЕНИЯ
Clip запрашивает только необходимые разрешения:
— Уведомления: для ежедневного дайджеста идей
— Интернет: для загрузки превью ссылок и AI-анализа
— Микрофон: для голосового ввода
